# pi-zero-tft-adapter

Eagle designs for a PCB to connect a Raspberry Pi Zero to an [Adafruit 2.2" TFT LCD](https://www.adafruit.com/products/1480).

| LCD Pin | RPi Pin | RPi Pin # |
|---------|---------|-----------|
| BL_LED  | GPIO_18 | 12        |
| SCK     | SCLK    | 23        |
| MISO    | MISO	  | 21        |
| MOSI	  | MOSI	  | 19        |
| CS	    | CE0	    | 24        |
| RST	    | GPIO_25	| 22        |
| D/C	    | GPIO_24	| 18        |
| Vin	    | 3.3V	  | 17        |
| GND	    | GND	    | 20        |

Pinout taken from http://aaron-brown.net/blog/?p=99
